# 프로젝트 문서 구조

이 폴더는 Brightstar 프로젝트의 개발 지침과 기획 문서를 포함합니다.

## 폴더 구조

- **`code-convetions.md`** - 코드 작성 규칙, 컨벤션, 스타일 가이드
- **`design-system.md`** - 디자인 가이드, UI/UX 명세, 컬러/타이포그래피 규칙
- **`development-guidelines.md`** - 기획 문서, 프로젝트 요구사항, 사이트맵 등
- ** `project-convetions.md`** - 프로젝트 컨벤션 문서

## 사용 방법

각 폴더에 관련 문서를 업로드하면 Claude가 개발 시 참고합니다.
